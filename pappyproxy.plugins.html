pappyproxy.plugins.tagcmds module¶

pappyproxy.plugins.tagcmds.clrtag(*args, **kwargs)¶

Clear all the tags from requests Usage: clrtag <request ids>

pappyproxy.plugins.tagcmds.tag(*args, **kwargs)¶

Add a tag to requests. Usage: tag <tag> [request ids] You can tag as many requests as you want at the same time. If no ids are given, the tag will be applied to all in-context requests.

pappyproxy.plugins.tagcmds.untag(*args, **kwargs)¶

Remove a tag from requests Usage: untag <tag> <request ids> You can provide as many request ids as you want and the tag will be removed from all of them. If no ids are given, the tag will be removed from all in-context requests.

pappyproxy.plugins.view.list_reqs(*args, **kwargs)¶

List the most recent in-context requests. By default shows the most recent 25 Usage: list [a|num]

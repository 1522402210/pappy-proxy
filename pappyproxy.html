add_cookie(cookie)¶

Add a pappyproxy.http.ResponseCookie to the response.

Warning

This will add duplicate cookies. If you want to add a cookie you’re not sure exists, use set_cookie()

add_data(data)¶

Used for building a response from a Twisted protocol. Add data to the response. The data must conform to the content encoding and transfer encoding given in the headers passed in to add_line(). Can be any fragment of the data. I do not suggest that you use this function ever.

Parameters:data (string) – The data to add
